Button Color Change

This project utilizes JavaScript to dynamically alter the background color of a web page upon clicking a button. The button, labeled "Change Background Color," triggers a JavaScript function that randomly generates a hexadecimal color code and applies it to the document's body element, effectively changing the background color of the entire page.
